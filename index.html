Pizzaria

O Pizzaria é um site criado para mostrar o cardápio de uma pizzaria de forma simples e bonita. Ele foi desenvolvido com HTML e CSS, seguindo as instruções da plataforma Alura e usando o site Marcelo como base. O objetivo é apresentar as pizzas e outros produtos de maneira clara e atrativa.

O design do site não é responsivo, ou seja, ele não se adapta bem a celulares e tablets, sendo mais indicado para quem acessa pelo computador. A paleta de cores foi escolhida para deixar o site com um clima acolhedor e gostoso, com tons quentes que combinam com o tema de uma pizzaria.

O tema do site é focado em uma pizzaria, destacando as opções de pizzas, promoções e informações sobre o que a loja oferece. Embora o design seja simples, ele é eficiente e fácil de usar, ajudando os visitantes a encontrar rapidamente o que procuram. O Pizzaria é uma boa maneira de mostrar os produtos de uma pizzaria de forma prática e visualmente atraente.

A Biblioteca da Meia-Noite

A Biblioteca da Meia-Noite é um site criado seguindo as instruções da plataforma Alura, usando o site Alura Books como base. O objetivo do site é apresentar o livro com o mesmo nome e dar aos usuários uma experiência legal e fácil de usar. O tema do site é inspirado no livro "A Biblioteca da Meia-Noite", que fala sobre escolhas e universos paralelos.

O design do site é responsivo, ou seja, ele funciona bem em celulares, tablets e computadores. As cores escolhidas para o site são suaves e acolhedoras, criando uma atmosfera tranquila e imersiva, que combina com o tema do livro. O uso de tons mais escuros e neutros ajuda a criar um ambiente misterioso, enquanto as cores mais claras destacam as informações e tornam a leitura agradável.

O site foi feito com HTML e CSS, mas também tem interatividade, o que permite que o usuário explore o conteúdo de maneira divertida. O layout foi pensado para deixar a navegação fácil e agradável, ajudando o visitante a aprender mais sobre o livro de uma forma simples e interessante.

Flashcards Gastronomia

O Flashcards Gastronomia é um site feito para ajudar as pessoas a aprender sobre gastronomia de forma fácil e prática. Foi desenvolvido seguindo as instruções da plataforma Alura e usando o site Flashcards como base. A ideia é permitir que os usuários aprendam sobre culinária com cartões de memória, que facilitam a revisão e o estudo de conceitos.

design do site é responsivo, ou seja, ele se adapta bem a diferentes tamanhos de tela, como celulares, tablets e computadores. As cores foram escolhidas para tornar o site agradável e fácil de usar, sem distrações. A disposição dos elementos é simples e clara, facilitando a navegação.

O tema do site é gastronomia, e ele traz conteúdo relacionado a culinária e técnicas de cozinha. O site foi feito com HTML para organizar o conteúdo, CSS para definir o estilo e JavaScript para adicionar interatividade, permitindo que os flashcards sejam usados de maneira dinâmica. O Flashcards Gastronomia é uma boa ferramenta para quem quer aprender mais sobre comida e culinária de uma forma divertida e eficaz.
